# Edomae Landing

## Environment variables
- `VUE_APP_API_URL` str: Url of the api, eg: `https://localhost`
- `VUE_APP_RECAPTCHA_SITE_KEY` str: recaptcha site key

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
